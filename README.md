# Platform Buddy

MAX更新から大会挑戦まで、1人で始めたリフターを支えるスマホ対応PWA試作アプリです。

## 主な機能

- PLAN / LOG / MAX / MEET の4タブ構成
- Buddyメソッド Lv1 / Lv2、5/3/1、HPS、Smolov Jr.風テンプレートのプラン作成
- プラン内でトップセット、バックオフ、バリエーション、アクセサリーの実績を複数セット記録
- RPE実績をもとにした翌週への補正コメント
- 疲労サインが強い場合の Buddy Alert とデロード週提案
- 自由記録、履歴、編集、日付ごとのまとめ表示
- e1RM、ボリューム、RPE、BIG3バランスの分析
- 基礎知識、公式ルールリンク、白判定クイズ
- Excel出力、JSONバックアップ、PWAインストール対応

## 今回の更新

- PLAN画面上部に「Buddyメソッドとは？」の折りたたみ学習カードを追加
- BuddyメソッドLv1/Lv2の違いを短く表示
- 今週の目的カードを追加し、蓄積期・現在地チェック・強化期・ピーキング・最終チェックの意味を表示
- 最終チェック付近に白9本の考え方を小さく表示
- RPE補正コメントを「注意」「警告」「デロード候補」に段階化
- HPS、5/3/1、Smolov Jr.は簡略テンプレートである注記を維持
- PWAキャッシュを `platform-buddy-v59` に更新

## 共有時の注意

記録データはユーザーの端末内に保存されます。URLや端末が変わる場合は、事前にアプリ内のデータ管理からバックアップを書き出してください。
